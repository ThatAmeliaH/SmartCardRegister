The software is designed to be as usable as possible without a mouse. To allow for this, the following keyboard shortcuts have been implemented:

**Appearance**
- Close software: **Escape**
- Toggle fullscreen: **F11**

**File Management**
- Save register as file: **Ctrl + S**
- Open register from file: **Ctrl + O**

**Student Management**
- Add new student to register: **Ctrl + N**
- Delete selected student from register: **Delete**
- Delete selected student from register (skips confirmation dialogue): **Ctrl + Delete**

**Student Presence Management**
- Set selected student to _PRESENT_: **Alt + 1**
- Set selected student to _LATE_: **Alt + 2**
- Set selected student to _ABSENT_: **Alt + 3**
- Toggle presence for selected student (time dependant): **Ctrl + Enter**
